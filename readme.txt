== Description ==

Simple Random Posts Widget provides you with a widget that displays random posts. You can set the number of random posts to be displayed... it's that easy

== Installation ==

1. Upload `simple-random-posts-widget.zip` to the `/wp-content/plugins/` directory and unzip
1. Activate the plugin through the 'Plugins' menu in WordPress
